Variables

Variables are used to store data in C++:

  • int → whole numbers (e.g., 20)
  • double → decimal numbers (e.g., 1.75)
  • char → single character (e.g., 'A')
  • string → text (e.g., "Alice")

Functions

Functions are reusable blocks of code that perform a task. In this example:

  • addNumbers(int a, int b) → takes two integers and returns their sum
  • Main program calls the function and prints the result
Sum: 15

Pointers

Pointers are variables that store memory addresses:

  • & → gives the address of a variable
  • * → gets the value stored at the address (dereference)
Value of num: 42
Address of num: 0x7ffeefbff4a8
Pointer stores: 0x7ffeefbff4a8
Value via pointer: 42
