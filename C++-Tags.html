C++ Tags / Keywords / Symbols Meaning Back

  • Data Types: int, float, double, char, bool, void, wchar_t — define variable and function return types.
  • Literals: 0, 1, 2 … (integers), 3.14 (floating point), 'a' (character), "hello" (string), true, false, nullptr — constant values used in programs.
  • Identifiers: Names like main, myVar, counter — user-defined names for variables, functions, classes.
  • Control Flow: if, else, switch, case, default, for, while, do, break, continue, goto — manage decision-making and loops.
  • Functions: return, void, inline, operator — define or return from functions. Example: return 0; exits a function (commonly main) and returns status to the OS.
  • Arithmetic Operators: +, -, *, /, % — perform basic math operations.
  • Relational Operators: ==, !=, <, >, <=, >= — compare values.
  • Logical Operators: &&, ||, ! — logical AND, OR, NOT.
  • Assignment Operators: =, +=, -=, *=, /=, %= — assign values to variables.
  • Increment/Decrement: ++, -- — increase or decrease by one.
  • Bitwise Operators: &, |, ^, ~, <<, >> — manipulate bits.
  • Other Operators: sizeof, typeid, new, delete, alignof — memory and type operations.
  • Semicolon ; — ends a statement (e.g., int x = 5;).
  • Braces { } — group statements into blocks (used in functions, loops, if-else).
  • Parentheses ( ) — used for functions and expressions.
  • Brackets [ ] — used for arrays and indexing.
  • Comma , — separates items in a list.
  • Colon : — used in switch labels (case 1:), class inheritance (class B : public A).
  • Scope Resolution Operator :: — defines scope (e.g., std::cout).
  • Arrow Operators -> and . — access members (pointer/object).
  • Ternary Operator ?: — shorthand conditional (e.g., x = (a > b) ? a : b;).
  • Preprocessor: #include, #define, #ifdef, #ifndef, #endif, #pragma — compiler instructions before compilation.
  • OOP Keywords: class, struct, public, private, protected, friend, virtual, this — define and access object-oriented code.
  • Memory Management: new, delete — dynamic allocation and deallocation.
  • Templates: template, typename — generic programming.
  • Exception Handling: try, catch, throw, noexcept — handle runtime errors.
