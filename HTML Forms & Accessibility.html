Labels for Accessibility

Labels connect text descriptions to input fields using for and id. Screen readers use labels to help visually impaired users understand the form fields.



<input type="text" aria-label="Search" placeholder="Search here"> <button aria-label="Submit search">Search</button>

ARIA Attributes

ARIA (Accessible Rich Internet Applications) attributes improve accessibility. Example: aria-label provides a label for screen readers without visible text.

<form> <label for="email">Email:</label> <input type="email" id="email" name="email" required> <label for="age">Age:</label> <input type="number" id="age" name="age" min="1" max="120"> <button type="submit">Submit</button> </form>

Form Validation

HTML5 provides built-in validation: - required → field must be filled - type="email" → must be a valid email - min / max → numeric range
