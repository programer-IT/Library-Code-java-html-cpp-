public class PrintExample { public static void main(String[] args) { System.out.println("Hello, World!"); System.out.print("This is "); System.out.print("on the same line."); } }

Explanation

System.out.println() prints text and moves the cursor to the next line, while System.out.print() prints text but keeps the cursor on the same line. That’s why "Hello, World!" is on one line, and "This is on the same line." continues without breaking.

Hello, World!
This is on the same line.
public class VariablesExample { public static void main(String[] args) { int age = 20; double gpa = 1.75; String name = "Alice"; boolean isStudent = true; System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("GPA: " + gpa); System.out.println("Is a student? " + isStudent); } }

Explanation

Variables store data. int is for whole numbers, double is for decimals, String is for text, and boolean is for true/false values. Printing them shows the stored values.

Name: Alice
Age: 20
GPA: 1.75
Is a student? true
import java.util.Scanner; public class InputExample { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Enter your name: "); String name = sc.nextLine(); System.out.println("Hello, " + name + "!"); } }

Explanation

To read user input, we use Scanner. nextLine() reads a full line of text typed by the user. Here, the name entered is combined with a greeting using string concatenation.

Enter your name:
Hello, Alice!
public class IfElseExample { public static void main(String[] args) { int score = 85; if(score >= 90) { System.out.println("Excellent!"); } else if(score >= 75) { System.out.println("Good job!"); } else { System.out.println("Keep trying!"); } } }

Explanation

The if-else statement allows decision making. If the first condition (score >= 90) is not true, the program checks the next one. Since 85 is greater than 75, "Good job!" is printed.

Good job!
public class SwitchExample { public static void main(String[] args) { int day = 3; switch(day) { case 1: System.out.println("Monday"); break; case 2: System.out.println("Tuesday"); break; case 3: System.out.println("Wednesday"); break; default: System.out.println("Other day"); } } }

Explanation

A switch is used when one variable is compared against many possible values. Here, day is 3, so the case 3 runs and prints "Wednesday". The break statement stops the switch after a match.

Wednesday
public class LoopExample { public static void main(String[] args) { for(int i=1; i<=3; i++) { System.out.println("Count: " + i); } } }

Explanation

A for loop repeats code a fixed number of times. Here, i starts at 1, increases by 1 each time, and stops when it becomes greater than 3.

Count: 1
Count: 2
Count: 3
public class ArrayExample { public static void main(String[] args) { String[] fruits = {"Apple", "Banana", "Cherry"}; for(String fruit : fruits) { System.out.println(fruit); } } }

Explanation

An array stores multiple values in one variable. The for-each loop is used to go through each element in the array automatically.

Apple
Banana
Cherry
public class MethodExample { static int square(int n) { return n * n; } public static void main(String[] args) { System.out.println("Square of 5: " + square(5)); } }

Explanation

A method is a block of code that can be reused. The method square() takes a number, multiplies it by itself, and returns the result. Methods make code cleaner and reusable.

Square of 5: 25
