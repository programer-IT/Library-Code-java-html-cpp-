All HTML Tags / Elements Meaning

Document Metadata Back

  • <!DOCTYPE> — Defines the document type.
  • <html> — Root element of HTML page.
  • <head> — Container for metadata.
  • <title> — Defines title in browser tab.
  • <base> — Sets default URL/base for relative links.
  • <link> — Links external resources (CSS, icons).
  • <meta> — Metadata (charset, viewport, SEO).
  • <style> — Internal CSS.
  • <script> — JavaScript code or external script.
  • <noscript> — Content if JS is disabled.
  • <template> — Client-side content not rendered.

Sections & Structure

  • <body> — Page content container.
  • <header> — Intro or navigation for a page/section.
  • <nav> — Navigation links.
  • <main> — Main unique content.
  • <section> — Thematic grouping of content.
  • <article> — Independent content (blog post, news).
  • <aside> — Sidebar/related content.
  • <footer> — Footer information.
  • <address> — Contact info.
  • <h1>...<h6> — Headings, from most to least important.
  • <p> — Paragraph.
  • <hr> — Horizontal line (section break).

Text & Inline Elements

  • <a> — Hyperlink.
  • <abbr> — Abbreviation or acronym.
  • <b> — Bold text (without extra importance).
  • <strong> — Strong importance (bold, semantic).
  • <i> — Italic text (no emphasis).
  • <em> — Emphasized text (semantic italics).
  • <mark> — Highlighted text.
  • <small> — Smaller text (fine print).
  • <span> — Inline container for styling.
  • <br> — Line break.
  • <wbr> — Word break opportunity.
  • <sub> — Subscript text.
  • <sup> — Superscript text.
  • <code> — Inline code.
  • <kbd> — Keyboard input.
  • <samp> — Sample output.
  • <var> — Variable in programming/math.
  • <q> — Short quotation.
  • <blockquote> — Long quotation.
  • <cite> — Reference to a source.
  • <del> — Deleted text (strike through).
  • <ins> — Inserted/added text.
  • <s> — Strikethrough (no longer relevant).
  • <u> — Underlined text.
  • <bdi> — Bi-directional isolation.
  • <bdo> — Bi-directional text override.

Lists

  • <ul> — Unordered list.
  • <ol> — Ordered list.
  • <li> — List item.
  • <dl> — Description list.
  • <dt> — Term in a description list.
  • <dd> — Description of the term.

Tables

  • <table> — Table container.
  • <caption> — Table title/caption.
  • <tr> — Table row.
  • <td> — Table cell.
  • <th> — Table header cell.
  • <thead> — Table head section.
  • <tbody> — Table body section.
  • <tfoot> — Table footer section.
  • <col> — Defines column properties.
  • <colgroup> — Groups table columns.

Forms & Input

  • <form> — Input form.
  • <input> — Input field.
  • <textarea> — Multiline text input.
  • <button> — Clickable button.
  • <select> — Dropdown list.
  • <option> — Option in a dropdown.
  • <optgroup> — Group of options.
  • <label> — Label for form control.
  • <fieldset> — Groups related inputs.
  • <legend> — Caption for fieldset.
  • <datalist> — List of pre-defined input options.
  • <output> — Calculation/result output.
  • <progress> — Progress indicator.
  • <meter> — Scalar measurement (like a gauge).

Media

  • <img> — Image.
  • <audio> — Audio content.
  • <video> — Video content.
  • <track> — Subtitles/captions for video/audio.
  • <map> — Image map.
  • <area> — Clickable area in image map.
  • <canvas> — Drawing graphics with JS.
  • <svg> — Scalable Vector Graphics.
  • <iframe> — Inline frame (embed another page).
  • <embed> — Embeds external application/media.
  • <object> — Embeds objects.
  • <param> — Parameters for <object>.
  • <picture> — Responsive images.
  • <source> — Media resource for <audio>/<video>.
