<label for="name">Name:</label> <input type="text" id="name" name="name" placeholder="Enter your name">

Text Input

<input type="text"> allows users to type a single line of text. The placeholder shows a hint inside the box.


<label for="pass">Password:</label> <input type="password" id="pass" name="pass" placeholder="Enter password">

Password Input

<input type="password"> hides the characters typed by the user for privacy.


<label><input type="checkbox" name="subscribe"> Subscribe to newsletter</label>
<label><input type="radio" name="gender" value="male"> Male</label>
<label><input type="radio" name="gender" value="female"> Female</label>

Checkbox & Radio

Checkbox allows multiple selections. Radio allows only one option from a group.



<label for="message">Message:</label>
<textarea id="message" name="message" rows="4" cols="40">Enter your message here</textarea>

Textarea

<textarea> allows multiple lines of text input. You can set rows and columns to adjust size.


<label for="country">Country:</label> <select id="country" name="country"> <option value="ph">Philippines</option> <option value="us">USA</option> <option value="jp">Japan</option> </select>

Select / Dropdown

<select> creates a dropdown list. Each <option> is a choice.


<form action="#" method="post"> <label for="email">Email:</label> <input type="email" id="email" name="email" placeholder="Enter your email">
<button type="submit">Submit</button> </form>

Form Submit

<form> groups input elements. action specifies where data is sent. method is usually GET or POST.
