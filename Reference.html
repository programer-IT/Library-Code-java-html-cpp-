Java — Keywords · Literals · Operators · Symbols · Constructs (Complete)

Reserved Keywords (classic & modern)

  • Keywords: abstract, assert, boolean, break, byte, case, catch, char, class, const (reserved), continue, default, do, double, else, enum, extends, final, finally, float, for, if, implements, import, instanceof, int, interface, long, native, new, package, private, protected, public, return, short, static, strictfp, super, switch, synchronized, this, throw, throws, transient, try, void, volatile, while.
  • Modern / Module / Pattern / Records: module, open, opens, requires, exports, transitive, provides, uses, to, var, yield, record, sealed, permits, non-sealed.

Data Types & Literals

  • Primitive types: byte, short, int, long, float, double, char, boolean.
  • Literals: integers (e.g., 0, 42), floats (3.14), char ('a'), string ("text"), boolean (true, false), null.
  • Numeric suffixes: L for long, F for float, binary/octal/hex literals (e.g., 0b1010, 0xFF), underscores in numeric literals (1_000_000).

Operators

  • Arithmetic: +, -, *, /, %.
  • Relational: ==, !=, <, >, <=, >=.
  • Logical: &&, ||, !.
  • Assignment: =, +=, -=, *=, /=, %=, and bitwise assignment variants.
  • Increment / Decrement: ++, -- (pre & post).
  • Bitwise: &, |, ^, ~, <<, >>, >>>.
  • Ternary: ? : (e.g., x = (a > b) ? a : b;).
  • Other: instanceof, method reference ::, lambda arrow ->.

Control Flow & Statements

  • Flow: if, else, switch, case, default, for, enhanced-for (for(Type x : arr)), while, do...while.
  • Flow control: break, continue, return (e.g., return 0; inside main), throw.

OOP / Classes / Interfaces

  • Declaration: class, interface, enum, record.
  • Inheritance / Implementation: extends, implements.
  • Access / Modifiers: public, protected, private, static, final, abstract, synchronized, volatile, transient, strictfp.
  • Object ops: new, this, super, instanceof.

Exceptions & Concurrency

  • Exceptions: try, catch, finally, throws, throw, assert.
  • Concurrency / Threads: synchronized, volatile, Thread API; high-level concurrency APIs (Executors, ForkJoin) are library-level.

Misc / Build & Organization

  • Packages & modules: package, import, module, requires, exports, opens, provides, uses, to, transitive.
  • Compilation-time: native (native methods), strictfp, transient, volatile.
  • Examples / syntax snippets:
    • public static void main(String[] args) { ... }
    • return 0; (common pattern in C-style; Java main returning int is only via System.exit(int) or in older patterns when main signature used int)
C++ — Keywords · Operators · Symbols · Constructs (Complete)

Core Keywords (classic & modern)

  • Types & control: asm, auto, bool, break, case, catch, char, class, const, consteval, constinit, constexpr, continue, default, delete, do, double, else, enum, explicit, export (rare), extern, float, for, friend, goto, if, inline, int, long, mutable, namespace, new, noexcept, nullptr, operator, private, protected, public, register (deprecated), reinterpret_cast (operator form), return, short, signed, sizeof, static, static_assert, static_cast, struct, switch, template, this, throw, try, typedef, typeid, union, unsigned, using, virtual, void, volatile, while.
  • Modern concurrency / coroutines / modules: thread_local, concept, requires, co_await, co_yield, co_return, module, import, export (module), using (alias).

Data Types & Literals

  • Fundamental: int, short, long, char, wchar_t, char16_t, char32_t, float, double, long double, bool, void.
  • Literals: integer, floating, char ('a'), string ("..."), boolean (true, false), pointer literal nullptr, user-defined literals.

Operators

  • Arithmetic: +, -, *, /, %.
  • Relational: ==, !=, <, >, <=, >=.
  • Logical: &&, ||, !.
  • Bitwise: &, |, ^, ~, <<, >>.
  • Assignment: =, +=, -=, *=, /=, %=, bitwise assignment variants.
  • Increment/Decrement: ++, --.
  • Pointer / member: * (dereference), & (address-of), ->, ..
  • Scope: :: (scope resolution, e.g., std::cout).
  • Ternary: ? :.
  • Cast operators: static_cast, dynamic_cast, const_cast, reinterpret_cast.

Punctuation & Structure

  • Statement terminator: ;
  • Block: { }
  • Parentheses: ( ) for calls/expressions
  • Brackets: [ ] for arrays
  • Comma: , separates expressions/parameters
  • Colon: : (labels, inheritance: class B : public A)

Preprocessor & Build

  • Preprocessor directives: #include, #define, #undef, #ifdef, #ifndef, #if, #else, #elif, #endif, #pragma.
  • Linkage/extern: extern "C", extern.

OOP / Memory / Templates

  • OOP: class, struct, union, public/private/protected, virtual, override (specifier), final (specifier).
  • Memory: new, delete, malloc/free (C API), smart pointers (std::unique_ptr, std::shared_ptr — library-level), sizeof.
  • Templates: template, typename, concepts (concept), using alias templates.
HTML — Elements · Attributes · Entities · Comments (Complete)

Document & Metadata

  • <!DOCTYPE> — Document type declaration (HTML5).
  • <html> — Root element.
  • <head> — Metadata container.
  • <title> — Page title.
  • <meta> — Metadata: charset, viewport, description, keywords.
  • <base> — Base URL for relative links.
  • <link> — External resources (CSS, icons).
  • <style> — Inline CSS.
  • <script> — JS code or external script file.
  • <noscript> — Fallback if JS disabled.
  • <template> — Client-side template not rendered until used.

Sections & Semantics

  • <body> — Main content.
  • <header> — Intro / nav area.
  • <nav> — Navigation links.
  • <main> — Primary unique content.
  • <section> — Thematic grouping.
  • <article> — Independent content piece.
  • <aside> — Sidebar/related content.
  • <footer> — Footer information.
  • <address> — Contact information.

Text, Inline & Grouping

  • <h1> ... <h6> — Headings from important to least.
  • <p> — Paragraphs.
  • <div> — Generic block container.
  • <span> — Inline container.
  • <a> — Anchor / link (href).
  • <img> — Image (src, alt, srcset).
  • <picture> & <source> — Responsive images.
  • <ul>, <ol>, <li> — Lists.
  • <table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot> — Tables.
  • <form> and controls: <input>, <textarea>, <button>, <select>, <option>, <label>, <fieldset>, <legend>, <datalist>, <output>.

Media & Graphics

  • <audio>, <video> — multimedia playback with <source> & <track> for captions.
  • <canvas> — scriptable bitmap drawing.
  • <svg> — Scalable Vector Graphics inline.
  • <iframe> — Embed another HTML document.
  • <object>, <embed> — External content (plugins/objects).

Text semantics & other inline tags

  • <strong>, <em>, <b>, <i>, <small>, <mark>, <del>, <ins>, <code>, <kbd>, <samp>, <var>, <cite>, <q>, <blockquote>.
  • Comments: <!-- comment -->
  • Entities & encoding: &nbsp;, &lt;, &gt;, &amp; and <meta charset="utf-8">.
